# Contributing

Danke für dein Interesse an der Mitarbeit!

## Prozess

1. **Fork** das Repository
2. **Create** eine Feature Branch (`git checkout -b feature/xyz`)
3. **Commit** deine Änderungen (`git commit -m 'Add feature'`)
4. **Push** zur Branch (`git push origin feature/xyz`)
5. **Pull Request** öffnen

## Standards

- Code Style: Projekt-Standards beachten
- Testing: Tests für neue Features hinzufügen
- Documentation: README aktualisieren, falls relevant
- Commits: Aussagekräftige Commit-Nachrichten

## Issues
Vor großer Arbeit: Issue erstellen und diskutieren.
