# WEBGUARDS-DE Organization Repository

Dieses Repository enthält gemeinsame Konfigurationen, Templates und Workflows für die gesamte WEBGUARDS-DE Organization.

## Struktur

### `/ISSUE_TEMPLATE/`
Standard-Vorlagen für verschiedene Issue-Typen:
- `bug.md` – Bug Reports
- `feature.md` – Feature Requests
- `task.md` – Tasks

### `/PULL_REQUEST_TEMPLATE.md`
Standard-Vorlage für Pull Requests

### Organization Settings
Dieses Repository wird vom GitHub verwendet, um Org-Level Settings anzuwenden:
- Default Issue und PR Templates
- Community Health Files (CODE_OF_CONDUCT, CONTRIBUTING, etc.)

## Verwendung

Änderungen an Templates wirken sich auf alle Repositories in der Organization aus (sofern nicht lokal überschrieben).
