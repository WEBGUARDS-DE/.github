# Code of Conduct

## Unsere Verpflichtung
Wir setzen uns für eine offene, respektvolle und professionelle Zusammenarbeit ein.

## Erwartungen
Alle Teilnehmer werden erwartet:
- Respektvolles und konstruktives Verhalten
- Faire Behandlung anderer
- Offenheit für Feedback
- Fokus auf gemeinsame Ziele

## Durchsetzung
Verstöße werden ernst genommen und angemessen gehandhabt.
